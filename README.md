# QWEN-to-SDXL Flow-Matching Adapter (Smoke Test)

## **Experimental ONLY**

This repository is highly experimental and **not** meant to be used for any practical application.

The focus of these experiments were to attempt mapping the unaligned feature space of a causal LLM (Qwen3.5-0.8B-Base in this case) to a _pre-trained_ UNet (stabilityai/stable-diffusion-xl-base-1.0).
